Git and Github

Git

Git is a powerful version control system that helps developers track and manage changes to their code. Think of it as a time machine for your projects, allowing you to save different versions, revert to previous states, and collaborate with others without overwriting each other's work. It's a distributed system, which means every developer has a complete copy of the project's history on their own computer, making it fast and reliable even without an internet connection.

GitHub

GitHub is a web-based platform that provides hosting for Git repositories. While Git is the underlying tool you use to manage your code, GitHub is a service that makes it easy to share, collaborate on, and manage those projects online. It's like a social network for developers, offering features like issue tracking, project management, and code review. GitHub is the largest host of source code in the world and is widely used by both individual developers and large companies.

Git and GitHub Commands to track and upload

1. git config --global user.name "Your name"

2. git config --global user.email "Your email"

3. git config --list --global

4. git init

5. git add .

6. git commit -m"name of the task"

7. continue 5 & 6 steps to add more tasks

8. git log --oneline (to check all the task thatis added)

9. git remote add origin

10. git branch -M main

11. git push -u origin main
